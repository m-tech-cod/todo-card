# Frontend Projects – HNG Internship (Stage 1)

This repository contains modern frontend UI components built with **HTML, CSS, and Vanilla JavaScript** as part of the HNG Internship Stage 1.

It includes:
- Todo Task Card (Stage 1a)
- Profile Card (Stage 1b)
- Landing Page (project entry point)

---

## Live Demo

https://m-tech-cod.github.io/todo-card/

---

## Projects Included

### Stage 1a – Todo Task Card
A modern interactive task management card featuring:
- Task completion toggle
- Priority levels (Low / Medium / High)
- Dynamic time remaining calculation
- Status updates (Pending → In Progress → Done)
- Edit and delete UI actions

---

### Stage 1b – Profile Card
A clean developer profile card featuring:
- Profile image and personal info
- Location display
- Social media links (GitHub, LinkedIn)
- Interactive follow/unfollow button
- Responsive modern UI design

---

### Landing Page (Home)
A simple entry page that acts as navigation hub between projects:
- Links to Todo Card project
- Links to Profile Card project
- Clean modern card-based UI
- Fully responsive design

---

##  Tech Stack

- HTML5 (semantic structure)
- CSS3 (modern UI, responsive design, animations)
- JavaScript (DOM interaction, state management)
