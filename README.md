# Todo Card Project

A modern, responsive Todo Task Card built with **HTML, CSS, and Vanilla JavaScript**.

## Live Demo


## Features

- Modern SaaS-style UI
- Responsive design (mobile & desktop)
- Task priority system (Low / Medium / High)
- Dynamic time remaining calculation
- Task completion toggle (checkbox)
- Status update (Pending → Done)
- Edit & Delete actions (demo)


## Tech Stack

- HTML5 (semantic structure)
- CSS3 (modern UI design, responsive layout)
- JavaScript (DOM manipulation, time logic)
